# Математическая библиотека для работы с размерами геометрических фигур
> Данная библиотека позволяет находить площади и периметр таких стандартных фигур, как [square---квадрат](квадрат), круг, прямоугольник и треугольник.

## Пример вызова функций

## сircle - круг
> Прозволяет найти площадь или периметр круга

### Площадь круга
    Принимает r(float):  радиус круга

    Возвращает: float Площадь круга по формуле pi*r^2

    сircle.area(r = 10) >> 314.1592653589793 

### Прериметр круга

    Принимает r(float):  радиус круга

    Возвращает: float Площадь круга по формуле 2*pi*r

    circle.perimeter(r = 10) >> 62.83185307179586 

## square - квадрат
> Позволяет найти Периметр и площадь квадрата

### Площадь квадрата

    Принимает a(float):  стронону квадиата

    Возвращает: float Площадь квадрата по формуле a^2

    square.area(a = 10) >> 100 

### Периметр квадрата

    Принимает a(float):  стронону квадиата

    Возвращает: float Площадь Периметр квадрата по формуле 4*a

    square.perimeter(a = 10) >> 44

## rectangle - прямоугльник
> Позволяет найти периметр и площадь прмоугольника

### Площадь прямоугольника

    Принимает a(float), b(float):  первую и вторую строноны прмясоугольника

    Возвращает: float Площадь прямоугольника по формуле a*b

    rectangle.area(a = 10, b = 12) >> 120 

### Периметр прямоугольника

    Принимает a(float), b(float):  первую и вторую строноны прмясоугольника

    Возвращает: float Периметр прямоугольника по формуле (a+b)* 2

    rectangle.perimeter(a = 10, b = 12) >> 44  

## Треугольник
> Позволяет найти периметр и площадь треугольника

### Площадь треугольника

    Принимает a(float),h(float):  основание и высоту треугольника

    Возвращает: float Площадь треугольника по формуле a *h / 2

    triangle.area(a = 10, h = 12) >> 60 

### Периметр треугольника

    Принимает a(float), b(float), c(float): три строноны треугольника

    Возвращает: float Периметр прямоугольнкиа по формуле a+b+c

    triangle.perimeter(a = 10, b = 12, c = 13) >> 35
