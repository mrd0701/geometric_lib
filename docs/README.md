# Математическая библиотека для работы с размерами геометрических фигур
    Данная библиотека позволяет находить площади и периметр таких стандартных фигур, как квадрат, круг, прямоугольник и треугольник.

## Пример вызова функций

### Круг
circle.area(r = 10) >> 314.1592653589793 

circle.perimeter(r = 10) >> 62.83185307179586 

### Квадрат
square.area(a = 10) >> 100 

square.perimeter(a = 10) >> 44

### Прямоугольник
rectangle.area(a = 10, b = 12) >> 120 

rectangle.perimeter(a = 10, b = 12) >> 44 

### Треугольник
triangle.area(a = 10, h = 12) >> 60 

triangle.perimeter(a = 10, b = 12, c = 13) >> 35
